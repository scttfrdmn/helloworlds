Honorable Mentions

Languages that didn't make the main collection, but deserve recognition

To qualify for the main gallery, a Hello World must use a distinct, general-purpose language that's stable, released, and actively used, not a variant, dialect, or platform-specific tool.

▶ What gets a language into the main collection?

To be included in the main gallery, a language must meet these criteria:

  • ✓ Distinct language: Not a variant, dialect, or compiler of an existing language
  • ✓ General-purpose or historically significant: Not limited to a single platform or application
  • ✓ Stable and released: Past 1.0 or widely adopted in production
  • ✓ Meaningful adoption: Has a community, documentation, and real-world usage
  • ✓ Not superseded: If a newer version exists in the collection, older versions are honorary
